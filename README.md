Este proyecto presenta un Dashboard en Excel que analiza una encuesta ficticia de intención de voto para las elecciones nacionales uruguayas. El análisis desglosa variables clave como segmentación etaria, nivel socioeconómico y distribución geográfica (Montevideo vs. Interior).

El diferencial del análisis es la inclusión de un KPI de Evolución Histórica, el cual contrasta la intención de voto actual con el comportamiento electoral de los bloques políticos desde la apertura democrática en 1985.
